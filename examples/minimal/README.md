# 최소 예시 — 치환 완료된 하네스의 밀도 기준

가상의 작업 *"legacy-utils 모듈 42개 파일을 TypeScript로 마이그레이션"* 에 대해
팩토리가 생성한 하네스의 핵심 2개 파일(HARNESS.md, state.json)을 치환 완료 상태로 보여준다.

구성자는 이 예시를 **톤과 밀도의 기준**으로 삼는다:

- 명령형·판정형 문장. 설명문 없음.
- 모든 정책이 수치로 확정됨 ("적절히" 없음).
- 커맨드는 복사-실행 가능.
- 나머지 파일(loops/, recovery/ 등)은 템플릿 구조가 동일하므로 생략했다. 실제 인도물에서는 전부 생성해야 한다 (CHECKLIST §1).
