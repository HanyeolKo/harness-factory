# 실전 예시 — 팩토리 프로토콜 1회 실제 수행 결과물

harness-factory의 구성자 프로토콜(Phase 0~4)을 **실제로 1회 수행**해 생성한 하네스다.
대상: harness-factory 레포 자체의 지속 개선 작업. 산출물 계약 13개 파일 전부가 치환 완료 상태로 들어 있다.

- Phase 1은 "알아서" 경로: 목적(Q1)만 사용자 요청으로 확정, 나머지는 기본값 일괄표 적용 (D-001에 기록)
- 대상 레포에 결정적 evaluator가 없어 "검증 스크립트 신설을 첫 백로그로" 규칙이 적용된 사례다 (state.json U-001)
- `examples/minimal/`이 톤·밀도 기준이라면, 이 예시는 **전체 구조와 파일 간 연결의 기준**이다
